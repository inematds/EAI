## Jogo 1: Ball Fusion Arena

### Conceito
Jogo estilo brick-breaker com elementos roguelike. O jogador lança bolas que quebram blocos, coletam power-ups e podem ser fundidas para criar bolas mais poderosas.

### Mecânicas Principais
- [ ] Sistema de lançamento de bolas com física
- [ ] Grid de blocos destrutíveis (tijolos coloridos)
- [ ] Power-ups que caem dos blocos
- [ ] Sistema de fusão: combinar 2 bolas iguais = bola melhor
- [ ] Tipos de bolas: Normal, Fogo, Gelo, Elétrica, Laser, Explosiva
- [ ] Upgrades permanentes entre runs
- [ ] Sistema de waves/níveis progressivos

### Características
- [ ] 6+ tipos de bolas com habilidades únicas
- [ ] 10+ power-ups diferentes
- [ ] Sistema de combo (quebrar múltiplos blocos)
- [ ] Boss battles a cada 10 níveis
- [ ] Leaderboard de pontuação
- [ ] Integração EAI (XP, moedas)

### Controles
- Mouse/Touch para mirar e lançar
- Arrastar para ajustar ângulo

## Jogo 2: Fashion Battle EAI

### Conceito
Jogo social de moda inspirado no viral "Dress to Impress" do Roblox. Jogadores criam looks baseados em temas e votam nos melhores.

### Mecânicas Principais
- [ ] Sistema de customização de personagem
- [ ] Guarda-roupa com roupas, acessórios, cabelos
- [ ] Temas aleatórios para cada rodada
- [ ] Timer para montar o look
- [ ] Sistema de votação (estrelas 1-5)
- [ ] Ranking e premiação

### Características
- [ ] 50+ peças de roupa iniciais
- [ ] 20+ acessórios (óculos, chapéus, joias)
- [ ] 15+ estilos de cabelo
- [ ] 10+ temas diferentes (Festa, Praia, Escola, Gótico, etc.)
- [ ] Moedas para desbloquear itens premium
- [ ] Skins sazonais/eventos

### Modos de Jogo
- [ ] Solo: Jogar contra NPCs
- [ ] Diário: Tema do dia com ranking global
- [ ] Desafio: Temas específicos com restrições

### Controles
- Click/Touch para selecionar itens
- Drag and drop para vestir

## Jogo 3: Plinko Tycoon

### Conceito
Jogo idle/casual onde o jogador solta bolas em um tabuleiro de pinos (plinko) para ganhar moedas e upgrades. Mecânica viciante e relaxante.

### Mecânicas Principais
- [ ] Tabuleiro de plinko com física realista
- [ ] Bolas que caem e coletam multiplicadores
- [ ] Slots no fundo com diferentes recompensas
- [ ] Sistema de upgrades para bolas e tabuleiro
- [ ] Auto-drop para gameplay idle
- [ ] Prestígio/Rebirth para multiplicadores permanentes

### Upgrades
- [ ] Velocidade de drop
- [ ] Quantidade de bolas simultâneas
- [ ] Multiplicadores de moedas
- [ ] Novos tipos de bolas (dourada, diamante)
- [ ] Pinos especiais (bounce+, magnet, split)
- [ ] Slots especiais (jackpot, bonus round)

### Características
- [ ] Efeitos visuais satisfatórios
- [ ] Sons de moedas e multiplicadores
- [ ] Sistema de conquistas
- [ ] Offline earnings
- [ ] Eventos especiais com tabuleiros temáticos

### Controles
- Click/Touch para soltar bola
- Botões para upgrades

## Jogo 4: Survivors EAI

### Conceito
Jogo de sobrevivência estilo Vampire Survivors. O jogador controla um personagem que ataca automaticamente enquanto desvia de hordas de inimigos e escolhe upgrades.

### Mecânicas Principais
- [ ] Movimento com WASD/joystick virtual
- [ ] Ataques automáticos
- [ ] XP dos inimigos para level up
- [ ] Escolha de upgrades a cada level
- [ ] Armas que evoluem e combinam
- [ ] Timer de sobrevivência (10-30 min runs)

### Armas/Habilidades
- [ ] Espada giratória
- [ ] Projéteis mágicos
- [ ] Escudo orbital
- [ ] Raio elétrico
- [ ] Explosão de área
- [ ] Invocação de aliados

### Inimigos
- [ ] Slimes (básico)
- [ ] Morcegos (rápidos)
- [ ] Esqueletos (resistentes)
- [ ] Fantasmas (atravessam obstáculos)
- [ ] Bosses a cada 5 minutos

### Características
- [ ] 3+ personagens jogáveis
- [ ] 10+ armas/habilidades
- [ ] Evoluções de armas (combinar 2 = arma suprema)
- [ ] Mapa procedural
- [ ] Sistema de meta-progressão
- [ ] Moedas para desbloquear personagens/itens

### Controles
- WASD ou setas para mover
- Touch: joystick virtual

## Jogo 5: Merge Kingdom

### Conceito
Jogo de fusão onde o jogador combina unidades iguais para criar unidades mais fortes, que batalham automaticamente contra inimigos.

### Mecânicas Principais
- [ ] Grid de batalha (5x5 ou 6x6)
- [ ] Spawn de unidades básicas
- [ ] Drag para mover unidades
- [ ] Merge: 2 iguais = 1 melhor
- [ ] Batalhas automáticas contra waves
- [ ] Boss battles

### Tipos de Unidades
- [ ] Guerreiro (melee, tanque)
- [ ] Arqueiro (ranged, DPS)
- [ ] Mago (AoE, magia)
- [ ] Curandeiro (suporte)
- [ ] Cada tipo tem 8 níveis de evolução

### Características
- [ ] Sistema de waves infinitas
- [ ] Progressão de dificuldade
- [ ] Moedas para comprar unidades
- [ ] Diamantes para acelerar/premium
- [ ] Conquistas e milestones
- [ ] Ranking de wave máxima

### Mecânicas Avançadas
- [ ] Sinergias entre tipos (3 arqueiros = bonus)
- [ ] Posicionamento estratégico
- [ ] Habilidades especiais das unidades
- [ ] Equipamentos/buffs temporários

### Controles
- Drag and drop para mover/fundir
- Botões para comprar unidades

## Especificações Técnicas

### Stack
- HTML5 Canvas
- JavaScript vanilla (sem frameworks)
- CSS para UI
- localStorage para saves

### Padrões
- Classes ES6 para organização
- requestAnimationFrame para game loop
- Touch e mouse support
- Responsivo para mobile

### Integração EAI
- Salvar progresso no localStorage
- postMessage para comunicar XP/moedas ao parent
- Formato padrão de dados do jogador
