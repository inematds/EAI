# Pesquisa: Jogos Arcade para Adolescentes (10-16 anos)

> Pesquisa realizada em 30/12/2024 para o portal EAI Games

---

## Objetivo

Identificar jogos arcade avançados e populares entre adolescentes de 10 a 16 anos que podem ser desenvolvidos com HTML5/JavaScript/Canvas para o portal EAI.

## Tecnologias Recomendadas

### Frameworks JavaScript para Jogos

- **Phaser.js** - Framework completo para jogos 2D
- **Kaboom.js** - Simples e fácil de usar, ideal para arcade
- **PixiJS** - Renderização 2D de alta performance
- **Three.js** - Para jogos 3D/WebGL como corridas
- **Canvas API nativo** - Para jogos mais simples

### Características Importantes

- Responsivo (funcionar em mobile e desktop)
- Controles por teclado E touch
- Sistema de pontuação e ranking
- Salvar progresso no localStorage
- Compartilhamento em redes sociais
